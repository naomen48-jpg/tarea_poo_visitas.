# Sistema de Registro de Visitantes

Aplicación de escritorio en Python con Tkinter que permite gestionar visitantes mediante operaciones CRUD.

## Funcionalidades
- Registrar visitantes
- Visualizar en tabla
- Eliminar registros
- Validación de campos

## Ejecución
python main.py
